# Joshua Polanco Portfolio Draft

This folder contains a simple digital portfolio draft designed for a class assignment and ready to upload to GitHub Pages.

## Files
- `index.html` — main portfolio page
- `styles.css` — styling for the page

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` and `styles.css`.
3. In the repository, go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save the settings.
7. GitHub will generate a public portfolio link.
